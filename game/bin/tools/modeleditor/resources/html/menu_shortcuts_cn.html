菜单快捷键

快捷键 动作 菜单项
F1 打开本页。 帮助→快捷键
Alt + F4 关闭应用程序。 文件→退出
Ctrl + A 添加所选择的模型到视窗，使得可以与当前的模型组合浏览 –当所编辑的模型属于SuperModel的一部分时非常有用。
请注意添加的模型是只能浏览无法改变的。 		文件→添加模型
Ctrl + B 用模型的动画重新生成模型的可视盒 –这是用来在客户端可见和生成阴影的盒子。 文件→重新生成显示盒
Ctrl + H 开启/关闭面板的显示。 视图→显示面板
Ctrl + O 打开打开模型对话框，你可以在对话框里指定一个.model文件来加载。 文件→打开模型
Ctrl + R 把文件恢复到上次保存的状态。 文件→取消所有操作
Ctrl + S 保存当前的模型以及其它被修改过的文件。 文件→保存
Ctrl + T 从磁盘中重新载入模型的所有贴图 – 这对于刷新修改过的贴图很有用。 文件→重新载入所有贴图
Ctrl + Y 恢复最近一次被撤销的动作。 编辑→恢复
Ctrl + Z 撤销最近一次操作。
要确保按下时鼠标在3D视窗内。
只有改变模型的动作（例如修改模型的材质或者增加/删除动画及LODs）才会被加入到恢复列表。
诸如在显示面板里的修改，或者改变摄像机视角或高度的动作是不会被加入到列表里的。 		编辑→撤销
