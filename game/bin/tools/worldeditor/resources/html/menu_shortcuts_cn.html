菜单快捷键

快捷键 动作 菜单项
F1 打开本页。 帮助→快捷键
Alt + F4 关闭应用程序。 文件→退出
Ctrl + A 选择空间内的所有对象。 编辑→全选
Ctrl + B 保存所选的对象为预制模型。 编辑→将选定的物体保存为预制模型。
Ctrl + D 取消选择所选择的对象。 编辑→取消选择
Ctrl + H 开启/关闭面板的显示。 视图→显示面板
Ctrl + K 重新载入所有分块。 文件→重新载入所有分块
Ctrl + R 重新载入所有贴图。 文件→重新载入所有贴图
Ctrl + S 进行快速保存。 文件→保存
Ctrl + T 将壳体保存为壳体模板。 编辑→将壳体保存为壳体模板
Ctrl + Y 恢复最近所撤销的对场景的改变。 编辑→恢复
Ctrl + Z 撤销最近对场景的改变。
只有改变了场景的操作（例如改变地形或操作对象）才会被加入到恢复列表中。
诸如改变笔刷强度或着改变对象对齐设置之类的操作不会被加入到列表中。 		编辑→撤销
Ctrl + Alt + S 进行全面地保存。 文件→处理数据
