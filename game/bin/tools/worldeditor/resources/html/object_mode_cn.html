对象模式

快捷键 动作
Alt 如果选择了至少一个对象：

激活缩放gizmo。

如果鼠标处于被拖拽状态：

取消选择被鼠标拖拽区域覆盖的对象。

Ctrl 如果选择了至少一个对象：

添加对象。

Shift 如果选择了至少一个对象：

开启旋转gizmo。

Delete 删除所选择的对象。
Enter 放置所选择的对象到鼠标的位置。
Esc 取消选择所选择的对象。
B 开启/关闭渲染对象的BSP。
也可以使用选项面板的显示下拉列表的Scenery→BSP选项。
C 复制所选择的对象。
R 如果当前选择了一个壳体：

重新创建所选择的壳体。

G 开启/关闭以对象面板的栅格对齐组合框里制定的精度对齐。
也可以使用栅格对齐组合框激活复选框。
V 如果当前选择了一个壳体：

复制所选择的壳体，并且通过匹配的入口连接两个壳体。
V必须在鼠标中键被按下同时或之前按下。

点击鼠
标左键 		选择对象。
按下鼠标左
键并拖拽 		如果选中后直接移动复选框被选择：

以鼠标拖拽的方向移动所选择的对象。

如果选中后直接移动复选框未被选择：

选择所有在拖拽范围内的对象。

点击鼠
标右键 		如果点击了一个模型、壳体或着粒子：

打开对象的右键菜单。

点击鼠
标中键 		如果当前选择了壳体：

通过一个匹配的入口连接两个壳体。
如果V键被按下：

复制所选择的壳体，并且通过匹配的入口连接两个壳体。
注意V必须在鼠标中键被按下同时或之前按下。

滚动鼠
标中键 		如果Shift键被按下：

旋转所选择的对象15度。
如果栅格对齐组合框激活复选框被选择：

以角度指定的数值旋转所选择的对象。
