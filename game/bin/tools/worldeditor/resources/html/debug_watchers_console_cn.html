调试（观察器）控制台（Ctrl + F7）激活

当选择时：

快捷键 动作
Esc 退出。
Backspace 回到上一级。
Enter 如果所选的节点不是叶子节点：

展开到下一级。

如果所选的节点是叶子节点：

编辑并打开观察器。

Home 返回到根节点。
Page Down 下移一行。
Page Up 上移一行。
Numpad +
（加号） 		向上滚动观察器列表。
Numpad -
（减号） 		向下滚动观察器列表。
0-9 选择第0-9项。
Alt + (0-9) 选择第40-49项。
Alt + Shift + (0-9) 选择第50-59项。
Ctrl + (0-9) 选择第20-29项。
Ctrl + Shift + (0-9) 选择第30-39项。
Shift + (0-9) 选择第10-19项。
Shift + Enter 显示修改记录。

当编辑时：

快捷键 动作
（修改观察器的值的方式：）
Esc 退出编辑模式。
Enter 接受给观察器输入的值。
Page Up +1
Page Down -1
Alt + Page Up +1000
Alt + Page Down -1000
Ctrl + Page Up +100
Ctrl + Page Down -100
Shift + Page Up +10
Shift + Page Down -10
