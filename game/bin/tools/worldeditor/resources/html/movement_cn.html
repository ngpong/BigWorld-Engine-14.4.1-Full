移动

快捷键 动作
Caps Lock 增加摄像机速度。
A 向左移动摄像机。
D 向右移动摄像机。
E 向下移动摄像机。
J 使用操纵杆控制。
K 使用键盘控制。
Q 向上移动摄像机。
S 向后移动摄像机。
W 向前移动摄像机。
Ctrl + 1
Ctrl + 2
Ctrl + 3
Ctrl + 4 		分别设置摄像机速度为慢、中、快及特快。
也可以由工具按钮、、、来设置。
Ctrl + I 循环设置可使用的摄像机速度。
也可以使用摄像机速度工具按钮群组：、、、。
按下鼠标右
键并拖拽 		按照鼠标移动的方向旋转摄像机视角。
如果是开启的：

按照鼠标移动方向移动摄像机。

滚动鼠
标中键 		前后移动摄像机。
如果空格键被按下：

根据鼠标滚动的方向增加或减小摄像机的速度。
摄像机的速度也可以由工具按钮、、、来设置。
