Particle Systems

Key Action
F3 Plays or spawns the current particle system (depending on whether it is currently stopped or playing).
You can also use the and toolbar buttons to play or spawn.
F4 Stops the current particle system.
This is equivalent to clicking the toolbar button.
F5 Pauses or resumes the current particle system.
This is equivalent to clicking the toolbar button.
Ctrl If clicking on a particle's sub-system check box:

Causes all other sub-subsystems to acquire the opposite value of current sub-system.

Delete Deletes the selected objects.
