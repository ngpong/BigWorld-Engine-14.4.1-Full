菜单快捷键

快捷键 动作 菜单项
F1 打开本页。 帮助→快捷键
Alt + F4 关闭应用程序。 文件→退出
Ctrl + H 开启/关闭面板的显示。 视图→显示面板
Ctrl + O 打开打开对话框，你可以执行一个新的粒子目录。
也可以点击粒子系统面板的按钮。 		文件→打开粒子目录
Ctrl + S 保存当前的粒子系统。
也可以点击工具栏的按钮。 		文件→保存粒子系统
Ctrl + T 重新载入粒子系统所用到的贴图。
用第三方软件修改的贴图只有在粒子系统被重新载入或者这个选项被选择时才会被可见。 		文件→重新载入所有贴图
Ctrl + Y 恢复最近一次被撤销的动作。 编辑→恢复
Ctrl + Z 撤销最近一次操作。 编辑→撤销
